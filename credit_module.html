信用资产查询接口

信用资产查询

query_credit_detail(account)

功能说明

查询指定信用账户的详细资产信息，包括总资产、负债、可用资金等。

参数

  • account (StockAccount): 信用资金账号对象

返回值

CreditDetail对象或None，包含信用账户的详细资产信息

示例代码

# 创建信用账户对象 account = StockAccount('1208970161', 'CREDIT') # 查询信用资产详情 credit_detail = xt_trader.query_credit_detail(account) if credit_detail: print(f"总资产: {credit_detail.total_asset}") print(f"总负债: {credit_detail.total_debt}") print(f"可用资金: {credit_detail.cash_available}") else: print("查询失败或账户无信用资产")

融资融券标的查询

query_credit_subjects(account)

功能说明

查询指定账户可进行融资融券交易的标的证券列表。

参数

  • account (StockAccount): 信用资金账号对象

返回值

CreditSubjects对象列表或None

示例代码

# 查询融资融券标的 subjects = xt_trader.query_credit_subjects(account) if subjects: for subject in subjects: print(f"股票代码: {subject.stock_code}") print(f"股票名称: {subject.stock_name}") print(f"融资标志: {subject.credit_flag}") print(f"融券标志: {subject.debt_flag}") print("-" * 30) else: print("无融资融券标的或查询失败")

可融券数据查询

query_credit_slo_code(account)

功能说明

查询指定账户的可融券数据，包括可融券数量、费率等信息。

参数

  • account (StockAccount): 信用资金账号对象

返回值

CreditSloCode对象列表或None

注意: 此接口返回的是当前可融券的具体数据，包括数量限制和费率信息。

示例代码

# 查询可融券数据 slo_codes = xt_trader.query_credit_slo_code(account) if slo_codes: for slo in slo_codes: print(f"股票代码: {slo.stock_code}") print(f"可融券数量: {slo.enable_amount}") print(f"融券费率: {slo.slo_ratio}") print(f"剩余数量: {slo.left_amount}") print("-" * 30) else: print("无可融券数据或查询失败")

标的担保品查询

query_credit_assure(account)

功能说明

查询指定账户的标的担保品信息，用于融资融券业务的担保品管理。

参数

  • account (StockAccount): 信用资金账号对象

返回值

CreditAssure对象列表或None

示例代码

# 查询标的担保品 assures = xt_trader.query_credit_assure(account) if assures: for assure in assures: print(f"股票代码: {assure.stock_code}") print(f"担保品标志: {assure.assure_flag}") print(f"折算率: {assure.assure_ratio}") print("-" * 30) else: print("无担保品数据或查询失败")

负债合约查询

query_stk_compacts(account)

功能说明

查询指定账户的负债合约信息，包括融资融券的具体合约详情。

参数

  • account (StockAccount): 信用资金账号对象

返回值

StkCompacts对象列表或None

示例代码

# 查询负债合约 compacts = xt_trader.query_stk_compacts(account) if compacts: for compact in compacts: print(f"合约编号: {compact.compact_id}") print(f"股票代码: {compact.stock_code}") print(f"合约类型: {compact.compact_type}") print(f"负债数量: {compact.debt_amount}") print(f"开仓日期: {compact.open_date}") print("-" * 30) else: print("无负债合约或查询失败")

约券相关接口

券源行情查询

smt_query_quoter(account)

功能说明

查询约券业务的券源行情信息，获取可约券的证券及其相关信息。

参数

  • account (StockAccount): 信用资金账号对象

返回值

券源行情数据列表或None

说明: 约券是指投资者向券商预约借入证券进行卖空交易的业务。

示例代码

# 查询券源行情 quoters = xt_trader.smt_query_quoter(account) if quoters: for quoter in quoters: print(f"股票代码: {quoter.stock_code}") print(f"可约数量: {quoter.available_amount}") print(f"约券费率: {quoter.smt_ratio}") print(f"最小约券单位: {quoter.min_amount}") print("-" * 30) else: print("无券源行情数据或查询失败")

约券合约查询

smt_query_compact(account)

功能说明

查询已签订的约券合约信息，包括合约状态、约券数量等详情。

参数

  • account (StockAccount): 信用资金账号对象

返回值

约券合约数据列表或None

示例代码

# 查询约券合约 compacts = xt_trader.smt_query_compact(account) if compacts: for compact in compacts: print(f"合约编号: {compact.compact_id}") print(f"股票代码: {compact.stock_code}") print(f"约券数量: {compact.smt_amount}") print(f"合约状态: {compact.compact_status}") print(f"签订日期: {compact.sign_date}") print(f"到期日期: {compact.expire_date}") print("-" * 30) else: print("无约券合约或查询失败")
