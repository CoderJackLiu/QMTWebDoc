交易接口模块

包含所有股票交易相关的API接口，支持同步和异步下单、撤单等操作。

  • order_stock() - 股票同步下单
  • order_stock_async() - 股票异步下单
  • cancel_order_stock() - 股票同步撤单
  • cancel_order_stock_async() - 股票异步撤单
  • cancel_order_stock_sysid() - 按合同编号撤单
